# eim.backend

## 1. 실행 명령어

### 1.1. dist로 build 후 실행

```yarn run dist-server```

### 1.2. 개발 서버 실행

```yarn run dev-server```

### 1.3. test 실행

```yarn run test```

### 1.4. build 실행

```yarn run build```

### 1.5. lint 검사 실행

```yarn run lint```
